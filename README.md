## شروع-پروژه
```bash
git clone https://github.com/erfan-karimii/Shop_CV
cd yt_clone
pip install -r requirment.txt
cd core
python manage.py runserver
```
در نظر داشته باشید ساخت یک محیط مجازی و استفاده از ان برای نصب پکیج ها همیشه ایده خوبی می باشد.
